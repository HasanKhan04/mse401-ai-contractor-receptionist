# Requirements

## Functional Requirements

| ID  | Requirement                  | Priority | Acceptance Criteria              |
| --- | ---------------------------- | -------- | -------------------------------- |
| FR1 | Answer incoming calls        | High     | Call answered within 3 seconds   |
| FR2 | Collect customer information | High     | Name and phone number recorded   |
| FR3 | Identify customer service request    | High     | Correct classification recorded  |
| FR4 | Schedule appointments        | High     | Appointment added to calendar    |
| FR5 | Answer FAQs                  | High     | Accurate response provided       |

## Non-Functional Requirements

| ID   | Requirement  | Acceptance Criteria           |
| ---- | ------------ | ----------------------------- |
| NFR1 | Availability | 24/7 operation                |
| NFR2 | Reliability  | 99% uptime                    |
| NFR3 | Scalability  | Support multiple calls per day without service degradation                |
| NFR4 | Performance  | Response time under 3 seconds |
